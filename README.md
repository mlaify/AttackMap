# AttackMap

Understand your system. Map your attack surface.

## What is AttackMap?

AttackMap analyzes your codebase to:
- Build a system architecture model
- Identify attack surfaces
- Generate realistic attack paths
- Provide actionable mitigations

## Why?

Most tools find vulnerabilities.

AttackMap shows how they connect.

## Usage

```bash
attackmap analyze .
```

Output
	•	Architecture summary
	•	Attack surface map
	•	Threat model (STRIDE)
	•	Attack paths
	•	Mitigation recommendations
